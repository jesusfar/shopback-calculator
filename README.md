# ShopBack Calculator

## How to use

```bash
$ ./shopback-calculator <action> <arg1> [<arg2>....]
```
## Available tasks:
### Signup Domain
```bash
$ ./shopback-calculator signup <domain>
```
### Signup Domain
```bash
$ ./shopback-calculator spend [<numeric_argument>....]
```
### Redeem Domain
```bash
$ ./shopback-calculator redeem <domain>
```
## Unit Test

1. Install phpunit with:

```bash
 $ composer install
 ```
2. Run unit test
```bash
 $ ./vendor/bin/phpunit
```
